指令
指令
作用
/EasyGuiShop correctYaml 修正所有的配置文件，会把必选项和可选项全部的补全并且删除所有多余的选项，注释不一定会保留，修正后不会自动重载，请手动重载
/EasyGuiShop correctDatabase 修正数据库文件，会把部分脏数据修复
/EasyGuiShop reloadPlugin 重载插件，效果等同于卸载掉本插件后重新安装本插件
/EasyGuiShop openGui 打开界面
/EasyGuiShop help 查看帮助
/EasyGuiShop openMui <玩家名字> 以玩家的身份打开界面
/EasyGuiShop createIcon <图标名字> 以主手持有的物品为原型创建一个图标，需要在主手持有物品
/EasyGuiShop setIconVaultPrice <图标名字> < 价格 > 设置图标的金币价格
/EasyGuiShop setIconItemPrice <图标名字> < 价格 > 设置图标的物品价格，需要主手持物品
/EasyGuiShop setIconPlayerPointsPrice <图标名字> < 价格 > 设置图标的点券价格
/EasyGuiShop setIconStock <图标名字> < 数量 > 设置图标的库存
/EasyGuiShop setIconName <图标名字> < 新名字 > 设置图标的名字
/EasyGuiShop deleteIcon <图标名字> 删除图标，会级联删除关于该图标的所有内容
/EasyGuiShop plusShopPopularity <店铺名字> 给店铺添加人气
/EasyGuiShop subtractShopPopularity <店铺名字> 给店铺减少人气
/EasyGuiShop setIconLimitTime <图标名字> < 时间 > 设置图标限制交易的时间，以创建的时间开始计算，单位为秒
/EasyGuiShop resetIconLimitTime <图标名字> 设置图标限制交易的时间为默认值，也就是不限制
/EasyGuiShop setIconSystem <图标名字> <true|false> 设置图标为系统补货（无限）
/EasyGuiShop setShopSystem <店铺名字> <true|false> 设置店铺全部商品为系统补货（无限）
/EasyGuiShop setGoodSystem <店铺名字> < 商品名字 > <true|false> 设置店铺内商品为系统补货（无限）
权限
权限
作用
EasyGuiShop. 指令名使用某指令的权限，例如 EasyGuiShop.help 代表使用指令 / EasyGuiShop help 的权限
EasyGuiShop.shopNameLength.6 玩家能创建的店铺的长度，6 可以换为其他的数字，大的数字会覆盖小的数字，不给此权限无法创建店铺，如果无权限，默认值为 0
EasyGuiShop.shopAmount.3 玩家能创建的店铺的数量，3 可以换为其他的数字，大的数字会覆盖小的数字，不给此权限无法创建店铺，如果无权限，默认值为 0
EasyGuiShop.setGoodVaultPrice 玩家使用金币作为商品货币的权限
EasyGuiShop.setGoodItemPrice 玩家使用物品作为商品货币的权限
EasyGuiShop.setGoodPlayerPointsPrice 玩家使用点券作为商品货币的权限
EasyGuiShop.teleportShopLocationInterval 玩家两次传到店铺坐标的间隔，小的覆盖大的，默认 0
EasyGuiShop.receiveVersionInformation 玩家在进入游戏后收到版本信息的权限
